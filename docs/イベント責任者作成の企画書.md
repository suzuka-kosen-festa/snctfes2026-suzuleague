| **内容/ルール** |
| 4チーム（１チーム​2〜​５人）に分かれて、各チームの持ち点１００ポイントを失わないように予想していく。１チーム５問の問題を行う。 【システム関係】 視聴者参加のための手法 QRコードを見せる．クラス委員を使って勧誘．各室長にアプリのリンクを送る． 参加者がスマホ画面を見ちゃうので対策考えておくこと ※モニターについて（イベント代表者） まず，学校から借りることはできませんでした（今年についてもそれ以降についても難しいものと考えて結構です）．そして，業者の方から借りるという案も出ましたが，「画面の反射という問題を解決するためにはそれ専用のモニターが必要になるが，かなりの重量があるため1イベントのためにこれを用意するのはあまり現実的ではない」との回答を頂き，レンタルは困難であると判断しました． 代替案として顧問の先生に提案されたのがプロジェクターの利用でした． ※プロジェクターの光を合わせると光量が強めになるそうです モニターと比べて破損のリスクが少なく，軽量であるためこちらを利用出来たらと考えております． とはいえ，電源の問題や外光による影響の度合い・対策についてまだ考慮できていないため，「モニターはないもの」としてアプリ開発を行っていただきたいです． |
